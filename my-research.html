Copyright, Creative Commons (CC) Licenses, Fair Use

Copyright

Copyright protects the work of creators by giving them exclusive rights to control its use. In New Zealand, the Copyright Act outlines these rights, helping creators of works such as books, music, and art to decide how their work is shared.

Copyright protection in New Zealand | Ministry of Business, Innovation & Employment. (2016). Govt.nz. https://www.mbie.govt.nz/business-and-employment/business/intellectual-property/copyright/copyright-protection-in-new-zealand/ ‚Äå

Creative Commons
  1. CC BY (Attribution): Share, remix, and use commercially with credit.
  2. CC BY-SA (Attribution-ShareAlike): Share, remix, use commercially, share derivatives under same terms.
  3. CC BY-NC (Attribution-NonCommercial): Share non-commercially with credit.
  4. CC BY-ND (Attribution-NoDerivs): Share with credit, no changes or commercial use.

About CC Licenses - Creative Commons. (2023, August 16). Creative Commons. https://creativecommons.org/about/cclicenses/

Fair Use

Fair Use is a copyright law concept permitting limited use of copyrighted material without permission. It balances creators' rights and public needs for purposes like criticism, education, etc. Factors include purpose, work type, amount used, and impact.

What Is Fair Use? (2016, September 7). Copyright Alliance. https://copyrightalliance.org/faqs/what-is-fair-use/

Search Engine Optimization (SEO)

5 ways to improve the visibility of your website on search engine
  1. Conduct Keyword Research:
    Identify relevant keywords and phrases that potential visitors might use to find your content.
  2. On-Page Optimization:
    Optimize your website's meta titles, meta descriptions, headers, and content with targeted keywords.
  3. High-Quality Content:
    Create valuable, informative, and engaging content that meets the needs of your audience and addresses their queries.
  4. Mobile Optimization:
    Ensure your website is mobile-friendly, as Google prioritizes mobile-responsive sites in its rankings.
  5. User Experience (UX):
    Enhance user experience by improving page load speed, navigation, and overall site structure.

Web Performance & Maintenance

5 Ways to Reduce Page Loading Time:
  1. Optimize Images: Compress, resize, and use modern formats like WebP. Implement lazy loading for images.
  2. Minimize HTTP Requests: Combine CSS and JavaScript, minimize external resources.
  3. Enable Browser Caching: Store static assets locally using browser caching.
  4. Content Delivery Network (CDN): Use a CDN to distribute content across servers in various regions.
  5. Optimize Code: Minify HTML, CSS, and JavaScript, consider deferring JavaScript execution.
5 Important Actions for Website Maintenance:
  1. Regular Backups: Schedule automated offsite backups for data and files.
  2. Software Updates: Keep CMS, plugins, themes, and server software up to date.
  3. Security Audits: Conduct regular security audits, implement measures like firewalls.
  4. Content Review and Updates: Continuously review and update website content.
  5. Performance Optimization: Monitor performance, conduct speed tests, and implement improvements.

Web Security

5 common cyber attacks (threats) and preventive measures against those attacks:
  1. Phishing Attacks:
    • Attack: Deceptive emails to steal data.
    • Prevent: Educate users, use email filters, enable multi-factor authentication (MFA).
  2. Malware Infections:
    • Attack: Malicious software compromises systems.
    • Prevent: Keep software updated, use antivirus software, train employees.
  3. DDoS Attacks (Distributed Denial of Service):
    • Attack: Overwhelm websites with traffic.
    • Prevent: Employ DDoS mitigation services, configure load balancers, use rate limiting.
  4. SQL Injection Attacks:
    • Attack: Exploit web app vulnerabilities to access databases.
    • Prevent: Use prepared statements, input validation, and regular code audits.
  5. Insider Threats:
    • Attack: Internal personnel causing harm.
    • Prevent: Implement least privilege access, monitor user activities, and conduct employee training on security.
